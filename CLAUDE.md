This repository contains a collection of prompts for image to image AI workflows (for example: Gemini 2.5 Image to Image). 

This repository aims to provide a library of image to image prompts for various purposes:

- Some are "boilerplate" prompts intended to convey common image editing instructions to seed editing apps 
- Others are more specific/niche prompts that the user finds helpful

You can help the user with the following activitiers:

- Editing/proofreading existing prompts 
- Creating new prompts based on user instructions 
- Ideating the creating of complementary prompts or new categories of prompts taking maximal advantage of the potential of image to image as a generative AI modality
